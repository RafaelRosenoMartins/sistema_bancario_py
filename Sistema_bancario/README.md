# Projeto de Sistema Bancario

Trabalho prático de POO aplicando os conceitos fundamentais da linguagem Python.

## Organizacao
- Pacotes para dividir as responsabilidades (contas, operacoes e sistema).
- Classes e Objetos para representar clientes e contas.
- Heranca para diferenciar Conta Corrente e Poupanca.
- Polimorfismo nos metodos de saque.
- Agregacao na classe Banco que gerencia as contas.
- Composicao entre Conta e Historico.
- Encapsulamento com atributos protegidos.

## Como rodar
Apenas execute o arquivo principal:
python main.py
